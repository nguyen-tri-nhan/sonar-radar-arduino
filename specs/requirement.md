# Project sonar radar

## IOT
Arduino uno
Sonar sensor HC-SR04
Chưa có mạch wifi hay bluetooth, tạm thời kết nối với laptop bằng dây USB
## BE
Quarkus
- Nhận data từ Arduino
## FE
React App
- Hiển thị data

## Scope
- Dùng sensor như 1 radar, có các khoảng cách 5m - 10m - 15m - 20m
- Gửi các tín hiệu về quarkus app
- Hiển thị lên màn hình thông qua webapp
- Màn hình radar ban đầu màu đen, khi phát hiện vật cản sẽ hiển thị vật cản đó trên radar với màu đỏ
